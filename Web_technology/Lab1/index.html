Разработка простой веб-страницы с использованием HTML5

World Wide Web (WWW, web, веб, Всемирная паутина) - это распределенная информационная система,
предоставляющая доступ к гипертекстовым документам по протоколу HTTP (HyperText Transfer Protocol).
WWW - это сетевая технология, построенная на клиент-серверной архитектуре и использующая
инфраструктуру Интеренет для взаимодействия между клиентом и сервером (рис. 1).
